# Anki Plugin Agent Notes

## Reference Locations
- Add-on documentation: `addon-docs`
- Anki source of truth: `anki`

## Workflow Expectations
- When debugging or designing changes, first create a clear plan and then implement by following that plan without needing a separate reminder.

## New Plugin Process
To build a new plugin, start by asking clarifying questions until the requirements are unambiguous. Then create a concrete plan, and implement the plan step by step.
